About Me

Born and raised in Mountain View, California, I spent over a decade in the heart of Silicon Valley before pursuing my undergraduate education in Computer Science at NIT Rourkela. As a U.S. citizen, this choice reflects my commitment to gaining rigorous technical training while building a global perspective on research and innovation.

I currently work as a Research Intern at the Indian Statistical Institute, Bangalore under Prof. Dr. Saroj K. Meher, where I investigate how classical signal processing techniques relate to deep learning systems. Specifically, I'm exploring connections between frequency-domain operations and the learned representations in modern computer vision architectures, connecting foundational DSP theory with contemporary ML approaches.

Beyond signal processing and vision, I maintain strong interests in network systems and security, having built projects ranging from P2P protocols to packet analysis tools. I plan to return to the U.S. for graduate studies in machine learning, computer vision, or networked systems. I am actively seeking summer 2026 research opportunities to deepen my contributions to these areas.

Selected Projects

Sparse Signal Recovery & Sampling Toolkit

Investigated computational complexity and optimization of frequency-domain transforms by implementing DFT/FFT algorithms from scratch (no external libraries) and developing a Compressed Sensing framework using Matching Pursuit for sparse signal recovery. Achieved 8x speedup over naive FFT for N=4096 signals through Cooley-Tukey optimization, with comprehensive test coverage (>90%) validating mathematical correctness.

Python NumPy SciPy CI/CD
ScapyTapper

ScapyTapper is a simple network analysis tool which analyses ipv4, ipv6, and arp packets. It can display relevant information such as the source and destination IP, protocol involved, port info, TCP flags, DNS queries, and HTTP methods. Arguments can be passed on the command line to allow users to filter based on protocol, interface, etc. Featured in a technical deep-dive article.

Python Scapy Networking
SwarmShare (P2P Protocol)

Designed a custom application-layer P2P file-sharing protocol inspired by BitTorrent to explore distributed systems and network protocol performance. Features multi-threaded concurrent downloads, SHA-256 integrity verification per 1MB chunk, and tracker-based peer discovery. Tested scalability under simulated peer churn scenarios.

Socket Programming Distributed Systems SHA-256 Multi-threading
AlgoRoute (Optimization Engine)

Built interactive routing engine with scratch implementations of Dijkstra’s and A* on OSM data. Simulates dynamic traffic and generates isochrones for emergency response zone analysis.

Graph Theory NetworkX OSMnx Streamlit
